# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-05-18

### Added

- Plugin tile icon (headphones + speech bubble + 🇵🇱/🇬🇧 flags) embedded in the package and shown in Jellyfin Dashboard → Plugins.

## [1.0.0] - 2026-05-18

### Added

- Initial release.
- Audio + subtitle language flag emoji rendered under card title.
- Configurable: toggle audio/subs, English=GB vs US flag.
- Supports 50+ languages via ISO 639-1/2.
- Tested on Jellyfin 10.10–10.11 (web client).
